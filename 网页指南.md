# 样式部分
- 如果个位上的数字是 1，就给左边加个空格（或者写个margin-left）
- 分和秒都是两位数（小时的十位数上不要 零 (*零点除外*)）
# 脚本部分
+ 闹钟
- 闹钟其实就是在时钟的基础上增加一个预定时间设置，指定的时间与当前的时间相同，那么就响

+ 秒表
- 秒表是没有时针的，所以切换为秒表的时候，就转成 (分 : 秒 : 毫秒)
